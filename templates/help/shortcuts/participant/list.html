• Esc : Navigate to Participants Home
  • A : Add a Participant (eq. to click)
  • ← : Select previous element
  • → : Select next element
  • Enter ↲ : Navigate to details view for the selected participant
  • Del : Delete selected participant (eq. to click or drop element)
  • PgUp ▲ : Go to previous page (eq. to < Prev button click)
  • PgDn ▼ : Go to next page (eq. to Next > button click)
