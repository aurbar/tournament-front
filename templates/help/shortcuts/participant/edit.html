• Esc : Navigate to Participants Home
  • L : Navigate to Participants list (eq. to click)
  • A : Add a Participant (eq. to click)
  • ← : Navigate to previous participant view (eq. to Details button click)
  • Del : Delete current participant (eq. to click or drop element)
  • Enter ↲ : Save current participant and send to server (eq. to click)
